# 🧠 A-eye: Preprocessing

This submodule contains the **preprocessing pipelines** for MRI data used in the A-eye project, including:

- Bias field correction  
- Intensity normalization  
- Eye region extraction and orientation alignment  

---

Part of the **A-eye** project — automated 3D MRI segmentation and morphometric feature extraction for eye and orbit atlas construction, published in [*PLOS ONE*](https://doi.org/10.1371/journal.pone.0352257).

Try the pipeline online at **[aeye.hevs.ch](https://aeye.hevs.ch)**, or see the main repository for the full project description, figures, citation info, and license details:  
👉 [A-eye main repository](https://github.com/jaimebarran/a-eye)
